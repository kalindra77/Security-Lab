# Hris Breach PII

Hari ini saya telah melaporkan suatu kasus data breach berupa database dump berisi informasi PII dan kredensial user dari aplikasi HRIS sebuah perusahaan, Awal mulanya saya sedang melakukan kegiatan rutin monitoring salah satu forum underground yang biasa digunakan oleh para threat actor untuk mempublish aksinya, namun saya teralihkan dengan sebuah postingan data breach dengan title yang menyebut salah satu perusahaan Indonesia, setelah saya selidiki ternyata data tersebut merupakan kebocoran 2 tahun yang lalu (2024) yang mana kemungkinan repost atau malah baru ter publish.

Saya coba untuk mengecek domain terkait, ternyata domain nya masih aktif namun tidak digunakan oleh aplikasi apapun, setelah saya telusuri lebih lanjut ternyata perusahaan tersebut kemungkinan memiliki dua domain TLD. Saya coba melaporkan dan mengkonfirmasi hal terkait dengan kontak yang saya dapat, ternyata aplikasi HRIS tersebut benar sudah tidak digunakan dan sekaligus memvalidasi bahwa breach ini merupakan hasil lama, namun yang disayangkan adalah pihak perusahaan tidak tahu menahu atau tidak menyadari terkait kebocoran ini karena belum ada laporan sebelumnya, padahal informasi yang bocor merupakan data PII detail yang sangat sensitif dan kredensial user aplikasi yang bisa disalahgunakan oleh pihak pihak yang tidak bertanggung jawab.
